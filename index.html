Hello, my name is Rory.

I'm a web developer of sorts living in London. I primarily use Ruby and the Rails and Sinatra frameworks, but also work with HTML, CSS and Javascript.

I work at ChilliBean as their Lead Developer and built ChilliVault and SohoSoHo with the team there. We've been able to use some pretty cool technologies at ChilliBean and our stack consists of various Ruby on Rails and Sinatra applications that are backed by CouchDB. We also do some BDD and TDD using Cucumber and RSpec.

Outside of work I try and spend as much time with my friends as possible doing things that don't involve a computer! I am trying to do more open source work though, and you can find anything I've done on my GitHub profile.

I also have a deep affection for all animated GIFs.
